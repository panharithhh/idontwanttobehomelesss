## What it covers
Build a decoder-only **Transformer** language model from scratch (trained on the
tinyshakespeare dataset):
- self-attention from first principles — queries/keys/values, scaled dot-product
- multi-head attention, residual connections, LayerNorm, the feed-forward block
- stacking blocks into a GPT and training it to generate text
